Add a PDF sharing button to the Jetpack Sharing module, thanks to the WP Post to PDF Enhanced plugin.

== Description ==

Extends the Jetpack plugin and allows you to add a PDF sharing button to the list of sharing services available under Settings > Sharing in your dashboard.

Important: for this plugin to work, you must activate [Jetpack](http://wordpress.org/plugins/jetpack/) first, and activate the Sharing module. Then install the [WP Post to PDF Enhanced plugin](https://wordpress.org/plugins/wp-post-to-pdf-enhanced/).

== Installation ==

1. Install the Jetpack plugin, connect the plugin to WordPress.com
2. Activate the Sharing module
3. Install the PDF Sharing for Jetpack plugin via the WordPress.org plugin repository, or via your dashboard
4. Activate the plugin
5. Go to Settings > Sharing, and drag the PDF button to the sharing area.
6. Enjoy! :)
